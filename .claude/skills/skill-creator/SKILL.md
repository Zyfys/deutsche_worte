# Skill Creator

Создай новый скил для этого проекта.

## Как создать скил

1. Создай папку `.claude/skills/<skill-name>/`
2. Внутри создай файл `SKILL.md`
3. В SKILL.md опиши:
   - Название и назначение скила
   - Когда активировать
   - Пошаговые инструкции что делать
   - Примеры использования

## Структура SKILL.md

```markdown
# Название скила

Короткое описание что делает скил.

## Когда использовать

Триггеры и условия активации.

## Инструкции

1. Шаг 1
2. Шаг 2
...

## Примеры

...
```

## Существующие скилы проекта

- `add-word` — добавить слово в БД
- `check-db` — проверить состояние БД
- `run-bot` — запустить бота
- `skill-creator` — этот скил, создание новых скилов
